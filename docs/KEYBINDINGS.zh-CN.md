# 快捷键绑定

本文档是 TUI 所识别的所有键盘快捷键的权威目录。绑定按照**上下文**分组 —— 即它们生效的焦点或模态状态。列在"输入框 （Composer）"下的绑定仅在输入框获得焦点时生效；列在"对话记录（Transcript）"下的绑定仅在对话记录获得焦点时生效；以此类推。

快捷键绑定目前尚不支持用户自定义配置 —— 该功能已规划在未来版本中实现（#436, #437）。本文档是未来配置文件覆盖功能将引用的约定。

## 全局（任意上下文）

| 快捷键 | 操作 |
|----------------------|---------------------------------------------------------------|
| `F1` 或 `Ctrl-?` | 切换帮助覆盖层 |
| `Ctrl-K` | 打开命令面板（斜杠命令查找器） |
| `Ctrl-C` | 取消当前轮次 / 关闭模态框 / 二次确认后退出 |
| `Ctrl-D` | 退出（仅在输入框为空时） |
| `Tab` | 循环切换 TUI 模式：Plan → Agent → YOLO → Plan |
| `Shift-Tab` | 循环切换推理强度：off → high → max → off |
| `Ctrl-R` | 打开恢复会话选择器 |
| `Ctrl-L` | 刷新 / 清屏 |
| `Ctrl-Shift-E` / `Cmd-Shift-E` | 切换文件树侧边栏 |
| `Ctrl-T` | 切换实时对话记录覆盖层 |
| `Esc` | 关闭最顶层模态框 · 取消斜杠菜单 · 关闭提示 |

## 输入框 （Composer）

编辑你将发送的消息。

| 快捷键 | 操作 |
|-----------------------------|---------------------------------------------------------|
| `Enter` | 发送消息（或执行斜杠命令） |
| `Alt-Enter` / `Ctrl-J` | 插入换行而不发送 |
| `Ctrl-U` | 删除到行首 |
| `Ctrl-W` | 删除前一个词 |
| `Ctrl-A` / `Home` | 移至行首 |
| `Ctrl-E` / `End` | 移至行尾 |
| `Ctrl-←` / `Alt-←` | 向前移动一个词 |
| `Ctrl-→` / `Alt-→` | 向后移动一个词 |
| `Ctrl-V` / `Cmd-V` | 从剪贴板粘贴（括号粘贴也会自动处理） |
| `Ctrl-Y` | 从 kill buffer 粘贴 |
| `↑` / `↓` | 遍历输入历史（也可用于选择弹窗/附件项） |
| `Ctrl-P` / `Ctrl-N` | 遍历输入历史（替代方式） |
| `Ctrl-S` | 暂存当前草稿（通过 `/stash list`、`/stash pop` 恢复） |
| `Alt-R` | 搜索提示历史（再次按 Alt-R 退出） |
| `Tab` | 斜杠命令 / `@`-提及补全（弹窗感知） |
| `Ctrl-O` | 打开外部编辑器编辑当前草稿 |

### `@` 提及

输入 `@<部分内容>` 打开文件提及弹窗。`↑`/`↓` 遍历条目，`Tab` 或 `Enter` 确认。`Esc` 隐藏弹窗。从 v0.8.10 起（#441），补全结果会按提及频率重新排序 —— 你常用且近期使用的文件会排在前面。

### `#` 快速添加（记忆功能）

当 `[memory] enabled = true` 时，输入 `# foo` 并按 `Enter` 将 `foo` 作为带时间戳的条目追加到你的记忆文件中，**不会**发起对话轮次。参见 `docs/MEMORY.md`。

## 对话记录（Transcript）（当对话记录获得焦点时）

| 快捷键 | 操作 |
|----------------------|-----------------------------------------------------|
| `↑` / `↓` / `j` / `k` | 滚动一行（v0.8.13+：输入框为空时裸方向键也滚动） |
| `PgUp` / `PgDn` | 滚动一页 |
| `Home` / `g` | 跳转到顶部 |
| `End` / `G` | 跳转到底部 |
| `Esc` | 将焦点返回输入框 |
| `y` | 将选中区域复制到剪贴板 |
| `v` | 开始 / 扩展可视选择 |
| `o` | 打开光标下的 URL（支持 OSC 8 的终端） |

## 侧边栏（Sidebar）（当侧边栏获得焦点时）

| 快捷键 | 操作 |
|----------------------|-----------------------------------------------------|
| `↑` / `↓` / `j` / `k` | 移动选择 |
| `Enter` | 激活选中项（打开 / 聚焦 / 取消） |
| `Tab` | 循环到下一个侧边栏面板（文件 → 任务 → 代理 → 待办） |
| `Esc` | 将焦点返回输入框 |

## 斜杠命令面板（按 `Ctrl-K` 或输入 `/` 后）

| 快捷键 | 操作 |
|----------------------|-----------------------------------------------------|
| `↑` / `↓` | 移动选择 |
| `Enter` / `Tab` | 执行 / 补全高亮命令 |
| `Esc` | 关闭面板 |

## 审批弹窗（当工具请求审批时）

| 快捷键 | 操作 |
|----------------------|-----------------------------------------------------|
| `y` / `Y` / `1` | 批准一次 |
| `a` / `A` / `2` | 批准全部（后续调用自动批准） |
| `n` / `N` / `d` / `3` / `Esc` | 拒绝 |
| `v` / `V` | 查看工具执行前后的差异对比 |

## 新手引导（首次运行流程）

| 快捷键 | 操作 |
|----------------------|-----------------------------------------------------|
| `Enter` | 进入下一步（欢迎 → 语言 → API → …） |
| `Esc` | 回退一步 |
| `1`–`5` | 选择语言（语言选择步骤） |
| `y` / `Y` | 信任工作区（信任步骤） |
| `n` / `N` | 跳过信任提示 |

## v0.8.13 审计说明

- **Ctrl-S 用于暂存，而非历史搜索。** 在此版本中已修正 —— `Alt-R` 是历史搜索。
- **已移除虚构的 `Alt+Up`。** "编辑最后一条排队消息"的绑定曾列在 README 中，但在键分发代码中从未存在。
- **输入框为空时裸 Up/Down 方向键可滚动对话记录（v0.8.13）。** 此前 `should_scroll_with_arrows` 开关被硬编码为 false，意味着裸方向键始终导航输入历史，即使输入框为空。虚拟终端（Ghostty、Codex、Kitty-protocol）用户尤其受影响，因为他们无法使用 Cmd+Up / Alt+Up 快捷键。
- **可配置的键映射（#436）和 `tui.toml`（#437）仍被推迟。** `TuiPrefs` 结构和加载器已存在于 `settings.rs` 中，但尚未在启动时连接。命名绑定注册表（使 `~/.deepseek/tui.toml` 可覆盖具体按键）仍然待定。
- **未发现其他失效的绑定。** 以上列出的其他所有快捷键均在 `crates/tui/src/tui/ui.rs`（键盘事件分发）或 `crates/tui/src/tui/app.rs`（模式 + 状态转换）中有对应的活跃处理程序。

# DeepSeek TUI — VS Code Extension

VS Code companion extension for [DeepSeek TUI](https://github.com/Hmbown/DeepSeek-TUI).

## Features

- **DeepSeek: Open Chat** (`Ctrl+Shift+D`) — open a chat panel for DeepSeek TUI.
- **DeepSeek: Ask About Selection** — query DeepSeek about the selected code.
- **DeepSeek: Apply to Selection** — let DeepSeek transform the selected code.
- **DeepSeek: Run Test** — run tests with DeepSeek as reviewer.

## Requirements

- VS Code 1.85.0 or later.

## Development

```bash
npm install
npm run compile
npm run package
```
